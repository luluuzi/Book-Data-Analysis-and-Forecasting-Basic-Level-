## 简介
&emsp;&emsp;使用Python完成此次图书数据处理项目，爬取了在2024-2026年当当网畅销图书榜单数据；经过数据清洗去重、特征工程处理后，利用可视化工具完成图书市场多维度数据分析；分别搭建时间序列、随机森林两种预测模型对未来图书榜进行预测，并通过量化指标完成模型性能对比评估，输出图书销量预测数据，可用于图书市场趋势分析与毕业设计学习。
## 文件目录说明
| 文件                   | 说明                                                                                               |
| :--------------------- | :------------------------------------------------------------------------------------------------- |
| 1 爬取年月2024-2026.py | 爬虫脚本，采集当当网2024-2026图书榜单原始数据                                                      |
| 2 清洗无去重.py        | 数据预处理，处理缺失值、过滤无效数据、剔除重复数据                                                 |
| 3.特征工程.py          | 特征构造，提取时间、品类、评分等建模特征                                                           |
| 4 可视化分析.ipynb     | 可视化分析，分别对图书类别、价格、有关销量评分、时间趋势、各属性相关性这几个方面进行分析，绘制图表 |
| 5 时间序列预测.py      | 时序预测模型，完成图书榜时序预测                                                                   |
| 6 随机森林预测.py      | 随机森林回归模型，实现榜单预测                                                                     |
| 7 模型评估.py          | 模型效果评估，计算MSE、MAE、R²等评价指标                                                           |
## 运行顺序
1. 运行爬虫脚本，采集当当网图书原始榜单数据，得到原始dangdang_bestsellers.csv数据集；
2. 执行数据清洗脚本，过滤脏数据，生成标准化数据集；
3. 运行特征工程脚本，构建模型所需特征，接着得到featured_dangdang_data.csv特征处理后的数据集；
4. 打开可视化分析.ipynb，根据得到的特征数据集绘制分析图表，完成数据探索与图表生成；
5. 分别执行时序、随机森林脚本，完成预测建模；
6. 运行模型评估脚本，对比两种模型预测精度。
